## рҹ“– Overview

**VISTARA** вҖ” *Vista (pemandangan) + Tara (bintang/melampaui)* adalah **Smart Tourism Intelligence Platform** berbasis AI yang dirancang untuk mengubah cara wisatawan menjelajahi destinasi wisata. Terinspirasi dari keindahan Danau Toba, VISTARA hadir sebagai solusi rekomendasi wisata cerdas dengan pendekatan **AI-First** yang scalable dan siap bersaing di tingkat nasional.

### рҹҺҜ Problem Statement

Wisatawan menghadapi tantangan dalam menemukan pengalaman wisata yang personal dan bermakna:

| Masalah | Dampak |
|---------|--------|
| вқҢ Pencarian berbasis keyword | Hasil tidak relevan dengan konteks |
| вқҢ Informasi tersebar | Sulit mendapatkan rekomendasi holistik |
| вқҢ Kurang personalisasi | Pengalaman wisata terasa generik |
| вқҢ Bahasa deskripsi tidak konsisten | Kesulitan membandingkan destinasi |

### рҹ’Ў Our Solution

VISTARA menggunakan pendekatan **Multi-AI Stack** yang menggabungkan berbagai teknik kecerdasan buatan:

<div align="center">
  
| AI Capability | Technology | Business Value |
|---------------|------------|----------------|
| рҹ”Қ **Semantic Search** | Sentence Transformers + FAISS | Memahami makna, bukan keyword |
| рҹҺҜ **RAG Pipeline** | Hugging Face + LLM | Rekomendasi akurat & kontekstual |
| рҹ§  **Vector Embeddings** | multilingual-e5-small | Representasi cerdas 384 dimensi |
| рҹ“Ҡ **Sentiment Analysis** | Transformers + PyTorch | Insight kepuasan wisatawan |
| рҹ‘ҒпёҸ **Computer Vision** | Vision Transformer (Final) | Identifikasi destinasi dari foto |
| вҡЎ **Real-time** | FastAPI + Redis | Respons < 100ms |
